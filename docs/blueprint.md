# **App Name**: InfoEc

## Core Features:

- Admin Authentication: Firebase Authentication for admin login with a demo account.
- CRUD Panel basico: Admin panel for managing data of 5 institutions.
- Data Input: Input fields for solvencia, liquidez, morosidad, and activos_totales for each institution.
- Data Persistence: Saving and updating institution data in Firestore.
- Institution Listing: Public listing of the 5 institutions (Bancos and Cooperativas).
- Interactive Dashboard: Dashboard with donut charts and bar graphs for each institution's key indicators.
- Real-time Updates: Real-time data updates using Firestore's onSnapshot listener.

## Style Guidelines:

- Primary color: Deep sky blue (#3399FF) for a corporate and trustworthy feel, reminiscent of financial institutions.
- Background color: Light cyan (#E0F7FA), providing a clean and professional backdrop that enhances data visibility.
- Accent color: Lime green (#A3E47F), strategically used for calls to action, data highlights, and visual emphasis, creating a vibrant and engaging user experience.
- Body and headline font: 'Inter', a grotesque-style sans-serif, known for its modern and objective appearance.
- Use clear and professional icons representing each financial indicator.
- Modern and clean dashboard layout with emphasis on data clarity and visual hierarchy.
- Subtle animations for data updates and transitions to enhance user engagement.
